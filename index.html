My Projects

Project 1: Threat Identification and Control Effectiveness Analysis

This project involved assessing a widely-used email application to determine whether existing security controls effectively mitigated known vulnerabilities. The analysis focused on identifying potential threats, mapping them to control measures, and evaluating control performance. The outcome demonstrated gaps in protection and provided recommendations for strengthening the application’s security posture.

Project 2: Monthly IT Audit Program for an Ecuador based Financial Cooperative

This project consisted of designing and executing a recurring audit program to strengthen governance and compliance in a financial cooperative. It included creating detailed process-level audit plans with defined activities, actions, and verifications. The audits enhanced visibility into operational practices and improved adherence to IT security and management standards.

Project 3: Comprehensive Audit Reporting on IT Procedures and Controls

This project focused on preparing comprehensive audit reports for annual IT audit programs. The reports evaluated IT procedures and controls against international standards such as ISO 27001 and ISO 22301, as well as national regulations. Special attention was given to business continuity and backup management, with actionable insights delivered to improve resilience and compliance.

Project 4: Security Information and Event Management (SIEM) Lab

This project involved the implementation of a SIEM laboratory using Wazuh to manage security agents on Windows 10 and Ubuntu Desktop systems within controlled VirtualBox environments. The lab provided hands-on experience in centralized event monitoring, log analysis, and incident detection, simulating real-world enterprise environments.

Project 5: Adversary Emulation Lab with Caldera and Docker

This project involved setting up a controlled laboratory for adversary emulation using Caldera and Docker. Reconnaissance attacks were simulated on Windows 10 and Ubuntu Desktop machines in VirtualBox to evaluate detection and response capabilities. The lab served as a practical exercise in red-team/blue-team methodologies and adversarial simulation for cybersecurity training.
