My Projects

Project 1: Implementing an Intrusion Detection System: A Comprehensive Approach

An intrusion detection system (IDS) is a software application or device that monitors and analyzes network or system activities for malicious or unauthorized activities. This script calculates the SHA-256 hash of all files in the file system and compares them to the known hashes stored in the known_hashes dictionary. If a calculated hash does not match a known hash, the script outputs an "Intrusion detected in file" message. This IDS provides a simple and straightforward example of how host-based intrusion detection works.

Marriott Hotel Chain: Lessons Learned from the 2016 Data Breach

A report on the factors contributing to the Marriott Hotel Chain's 2016 data breach, including the company's failure to invest in its IT departments, inadequate risk management practices, and lack of a risk-aware culture. The report discusses the importance of conducting IT risk assessments and implementing frameworks to manage and oversee risk. The conclusion emphasizes the need for a proactive approach to cybersecurity and ongoing risk assessments to prevent future data breaches.
